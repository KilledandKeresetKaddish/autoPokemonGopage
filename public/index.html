本月看点

6 月下旬到 7 月中旬的主线很清楚：先完成社区日与团战轮换，随后把资源转向十周年庆典、传奇之路每日团战、GO Fest 2026:全球，最后收尾雷丘超级团战日。

  • 6/20 凉脊龙社区日：闪光概率提升、捕捉星尘 ×3、捕捉糖果 ×2；冰 / 龙系资源优先级最高。
  • 6/23–6/29 飞行计程车与6/25–6/29 被接管重叠：怒鹦哥（Squawkabilly）四种羽色首次登场，同期有火箭队限时调查与营救暗影莱希拉姆。
  • 6/24 起 5★切到纸御剑 / 铁火辉夜，超级团战切到超级大比鸟；6/27 盔甲鸟超级团战日继续需要火 / 电队。
  • 7/4–7/6 十周年庆典：捕捉 XP ×4、捕捉星尘 ×4，金币袋或黄金诱饵可遇十周年硬币索财灵；付费限时调查含特殊背景索财灵遭遇。
  • 7/6–7/10 传奇之路每天拆成独立团战准备窗口：7/7 白酋雷姆，7/8 黑酋雷姆，7/9 王者苍响 / 藏玛然特，7/10 起源双龙与原始双神。
  • 7/11–7/12 GO Fest 2026:全球：7/11（周六）超级超梦 X（超能 / 格斗，弱飞行 / 幽灵 / 妖精）、7/12（周日）超级超梦 Y（超能，弱虫 / 幽灵 / 恶）——两天需要不同的 Counter 队伍；同场还有捷拉奥拉登场。官方社区庆典弹窗名单覆盖台北、东京、首尔、曼谷、纽约、温哥华、汉堡、墨尔本等城市。
  • 7/18 雷丘超级团战日：超级雷丘 X 与 Y 同时首次登场，两者均弱地面；闪光概率提升窗口，集中地面系攻手（固拉多、土地云、龙头地鼠等）。

本月 Weekly Rotations

本期优先级（6/19 起）

先把 6/20 凉脊龙社区日当作星尘与糖果窗口；随后为 6/27 盔甲鸟超级团战日、7/4 十周年庆典、7/6–7/10 传奇之路每日团战和 7/11–7/12 GO Fest 全球预留票券、远程名额与补给。完整逐 Boss Counter 请看「当前团战 Counter」。

接下来要做什么

  • 6/20 14:00–17:00 凉脊龙社区日：闪光、星尘、糖果与糖果 XL 集中在 3 小时内，建议开星星碎片并提前清盒。
  • 6/24 后 5★切纸御剑 / 铁火辉夜，超级切大比鸟；6/27 盔甲鸟超级团战日是本月第二个重点闪光窗口。
  • 7/4–7/6 十周年庆典叠捕捉 XP ×4 与星尘 ×4；金币袋 / 黄金诱饵可刷十周年硬币索财灵，付费限时调查给特殊背景索财灵。
  • 7/6–7/10 传奇之路每天 18:00–20:00 都有团战目标，尤其 7/10 起源帝牙卢卡 / 起源帕路奇亚接原始盖欧卡 / 原始固拉多，提前规划本地队伍。

GO Fest 超级超梦 X/Y — 两天两套队伍

7/11 周六 超级超梦 X　　弱

超级超梦 X 是超能 + 格斗双属性，弱点覆盖飞行、幽灵、妖精。优先上勇士雄鹰、暗影路卡利欧（飞行系 / 妖精系快招）、耿鬼（幽灵系）。

7/12 周日 超级超梦 Y　　弱

超级超梦 Y 纯超能属性，弱虫、幽灵、恶——与常规超梦相同。班基拉斯、三首恶龙、耿鬼等恶 / 幽灵队主力可直接复用。两天需要不同队伍，提前各组一队！

练哪类攻手

  • 近期团战连续吃火 / 电：超级巨钳螳螂、纸御剑、铁火辉夜、盔甲鸟与超级大比鸟都能复用火系或电系队伍。
  • 奈克洛兹玛与后续拂晓之翼 / 黄昏之鬃形态需要恶 / 幽灵 / 虫方向；已成型的班基拉斯、三首恶龙、耿鬼可继续保留——GO Fest 超级超梦 Y 也用同一套。
  • 7/10 原始盖欧卡 / 原始固拉多分别需要草 / 电与水 / 草方向；传奇之路远程团战无上限期间适合集中补图鉴和 XL。
  • 7/11 超级超梦 X 需飞行 / 妖精方向——勇士雄鹰、暗影波克基斯、哲尔尼亚斯；与超梦 Y 方向不同，务必分开组队。
  • 7/18 雷丘超级团战日：超级雷丘 X/Y 同时首次登场，两者均弱地面 ；固拉多、土地云化身、龙头地鼠、超甲狂犀都是优选，闪光概率提升，提前准备地面队。
  • GO Fest 全球的免费内容很重：超级超梦 X/Y、捷拉奥拉、每日最多 9 张免费团战入场券、每天最多 6 次特殊交换；有官方社区庆典弹窗的城市可优先 RSVP。
