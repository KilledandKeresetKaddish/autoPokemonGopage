本月看点

6/24 团战大轮换已到位：本周先收尾轮换并把火 / 电队备好，随后把资源转向七月主线：十周年庆典、传奇之路每日团战、GO Fest 2026:全球，月末再到雷丘超级团战日。

  • 本周（6/22–6/28）Max 团战：极巨化咕咕登场，趁本周刷满极巨方块与队伍练度；下个 Max 周一高峰 6/29 换极巨化豆豆鸽。
  • 6/24 起团战大轮换：5★ 切到纸御剑 / 铁火辉夜（至 6/30），超级团战切到超级大比鸟；1★/3★ 同步切换为飞行主题（青绵鸟、姆克儿、蓝鸦、洗翠勇士雄鹰、下石鸟、秃鹰丫头），呼应飞行计程车活动。整月的暗影帝牙卢卡仍在。
  • 6/23–6/29 六月节 Arraiá（巴西限定）：全巴西可遇带特殊背景的水之品种帕底亚肯泰罗，戴牛仔帽的卡比兽闪光概率提升。
  • 6/23–6/29 飞行计程车与6/25–6/29 被接管重叠：怒鹦哥（Squawkabilly）四种羽色首次登场，同期有火箭队限时调查与营救暗影莱希拉姆。
  • 6/27 盔甲鸟超级团战日需要火 / 电队。
  • 7/4–7/6 十周年庆典：捕捉 XP ×4、捕捉星尘 ×4，金币袋或黄金诱饵可遇十周年硬币索财灵；同日 泪眼蜥社区日（7/4 14:00–17:00）。
  • 7/6–7/10 传奇之路每天一个团战准备窗口：7/7 白酋雷姆，7/8 黑酋雷姆，7/9 苍响 / 藏玛然特，7/10 起源双龙与原始盖欧卡 / 固拉多。
  • 7/11–7/12 GO Fest 2026:全球：7/11（周六）超级超梦 X（超能 / 格斗，弱飞行 / 幽灵 / 妖精）、7/12（周日）超级超梦 Y（超能，弱虫 / 幽灵 / 恶）——两天需要不同的 Counter 队伍。
  • 7 月 5★ 轮换：7/1–7/5 与 7/13–7/14 急冻鸟 / 闪电鸟 / 火焰鸟，7/15–7/21 盖欧卡，随后索尔迦雷欧、酋雷姆接力。
  • 7/18 雷丘超级团战日：超级雷丘 X 与 Y 同时首次登场，两者均弱地面；闪光概率提升窗口，集中地面系攻手（固拉多、土地云、龙头地鼠等）。
  • 8 月预告：8/8 孵化日、8/16 社区日、8/22 团战日，外加「选择你的道路:毒与藤蔓」（8/11–8/17）；具体宝可梦与加成官方公布后再补。
  • 长期线下活动：「宝可梦 GO × MLB 美职棒」赛季贯穿至 9/30，当天球场内可遇带场地背景的皮卡丘团战；「宝可梦化石博物馆:芝加哥」特展至 2027/4/11，现场可遇考古皮卡丘与超级化石翼龙。两者都在「长期活动」带。

本月 Weekly Rotations

本期优先级（6/24）

本周（6/22–6/28）Max 周一极巨化咕咕登场，本周先刷极巨方块；6/24 团战大轮换已到位，5★ 换纸御剑 / 铁火辉夜、超级换大比鸟、1★/3★ 转为飞行主题。随后为 6/27 盔甲鸟超级团战日、7/4 十周年庆典、7/6–7/10 传奇之路每日团战和 7/11–7/12 GO Fest 全球预留票券、远程名额与补给。完整逐 Boss Counter 请看「当前团战 Counter」。

接下来要做什么

  • 本周 Max 团战是极巨化咕咕（6/22–6/28），本周集中刷极巨方块、补极巨攻手练度。
  • 6/24 团战大轮换已到位：5★ 纸御剑 / 铁火辉夜（至 6/30）、超级大比鸟；1★/3★ 转为飞行主题（需电 / 冰 / 岩队）。暗影帝牙卢卡整月可打；6/27 盔甲鸟超级团战日是近期重点闪光窗口。
  • 7/4–7/6 十周年庆典叠捕捉 XP ×4 与星尘 ×4；金币袋 / 黄金诱饵可刷十周年硬币索财灵，付费限时调查给特殊背景索财灵。
  • 7/6–7/10 传奇之路每天 18:00–20:00 都有团战目标，尤其 7/10 起源帝牙卢卡 / 起源帕路奇亚接原始盖欧卡 / 原始固拉多，提前规划本地队伍。

GO Fest 超级超梦 X/Y — 两天两套队伍

7/11 周六 超级超梦 X　　弱

超级超梦 X 是超能 + 格斗双属性，弱点覆盖飞行、幽灵、妖精。优先上勇士雄鹰、暗影路卡利欧（飞行系 / 妖精系快招）、耿鬼（幽灵系）。

7/12 周日 超级超梦 Y　　弱

超级超梦 Y 纯超能属性，弱虫、幽灵、恶——与常规超梦相同。班基拉斯、三首恶龙、耿鬼等恶 / 幽灵队主力可直接复用。

两天需要不同队伍，提前各组一队！

练哪类攻手

  • 近期团战连续吃火 / 电：超级巨钳螳螂、纸御剑、铁火辉夜、盔甲鸟与超级大比鸟都能复用火系或电系队伍。
  • 奈克洛兹玛与后续拂晓之翼 / 黄昏之鬃形态需要恶 / 幽灵 / 虫方向；已成型的班基拉斯、三首恶龙、耿鬼可继续保留——GO Fest 超级超梦 Y 也用同一套。
  • 7/10 原始盖欧卡 / 原始固拉多分别需要草 / 电与水 / 草方向；传奇之路远程团战无上限期间适合集中补图鉴和 XL。
  • 7/11 超级超梦 X 需飞行 / 妖精方向——勇士雄鹰、暗影波克基斯、哲尔尼亚斯；与超梦 Y 方向不同，务必分开组队。
  • 7/18 雷丘超级团战日：超级雷丘 X/Y 同时首次登场，两者均弱地面 ；固拉多、土地云化身、龙头地鼠、超甲狂犀都是优选，闪光概率提升，提前准备地面队。
  • GO Fest 全球的免费内容很重：超级超梦 X/Y、捷拉奥拉、每日最多 9 张免费团战入场券、每天最多 6 次特殊交换；有官方社区庆典弹窗的城市可优先 RSVP。
