本月看点

7/1 更新：七月团战轮换已切到关都三神鸟、超级路卡利欧与暗影帕路奇亚；`data/raw/raids.json` 仍显示 6 月末纸御剑 / 铁火辉夜与超级大比鸟，因此本页本期推荐与 Counter 以已定稿的日历事件和 rotations 为准。

  • 7/1–7/5 急冻鸟・闪电鸟・火焰鸟 5★ 团战：三神鸟同池，7/1 18:00–19:00 还有关都三神鸟团战时刻；岩石系对急冻鸟 / 火焰鸟效率最高，电系可兼顾急冻鸟与闪电鸟。
  • 7/1–7/5 超级路卡利欧：弱火、格斗、地面；它也是后续钢 / 格斗糖果加成的高价值 Mega，值得补能量。
  • 7/1–8/4 暗影帕路奇亚：周末暗影团战长期目标，龙 / 妖精队准备好；暗影形态输出价值高但成本也高。
  • 至 7/3 选择你的道路:炽热余烬：先清限时调查；火 / 电主题资源能衔接本周鸟类与超级路卡利欧克制队。
  • 7/2 波波聚焦时刻：18:00–19:00 搭配进化经验 ×2，适合囤低糖进化与幸运蛋。
  • 7/4–7/6 十周年庆典叠捕捉 XP ×4 与星尘 ×4；同日 泪眼蜥社区日 14:00–17:00 有闪光提升与孵化距离 1/4。
  • 7/6–7/10 传奇之路每天换团战重点：7/7 白酋雷姆 / 捷克罗姆 / 拂晓之翼奈克洛兹玛，7/8 黑酋雷姆 / 莱希拉姆 / 黄昏之鬃奈克洛兹玛，7/9 王者苍响 / 藏玛然特，7/10 起源双龙与原始双神。
  • 7/11–7/12 GO Fest 2026:全球：超级超梦 X / Y 分日登场，捷拉奥拉特殊调查、每日最多 9 张免费团战券与 6 次特殊交换是核心收益。
  • 7/18 雷丘超级团战日、7/21 威洛博士生日调查、7/25–7/26 臭氧攀升是七月后段重点；臭氧攀升给陨石与烈空坐遭遇，想做超级烈空坐请预留资源。

本月 Weekly Rotations

本期优先级（7/1）

今天起以七月定稿日历为准：先打关都三神鸟、超级路卡利欧与暗影帕路奇亚；`data/raw/raids.json` 仍停在 6 月末，所以当前团战 Counter 未采用其过期 5★/Mega 列表。7/4 十周年庆典与泪眼蜥社区日会叠高收益，7/6–7/12 则进入传奇之路与 GO Fest 高强度团战周。

今天先做

  • 7/1 18:00–19:00 关都三神鸟团战时刻：急冻鸟 / 火焰鸟优先岩石队，闪电鸟优先岩石或冰队。
  • 超级路卡利欧只开到 7/5，补能量价值高；火、格斗、地面队都能打，后续也能为钢 / 格斗系捕捉加糖。
  • 暗影帕路奇亚是整月周末暗影目标，龙 / 妖精队准备好；缺人时优先组队，不建议硬单。

7/4 双重点

  • 十周年庆典：捕捉 XP ×4 与星尘 ×4，适合开幸运蛋 / 星星碎片集中捕捉；金币袋或黄金诱饵可刷索财灵。
  • 泪眼蜥社区日：14:00–17:00 闪光提升，孵化距离 1/4；提前清蛋位并备好孵化器。
  • 同日活动重叠，建议先抓社区日，再把剩余时间给周年庆捕捉与索财灵。

团战周资源

  • 7/6–7/10 传奇之路每天有不同传说 / 超级重点，远程票和绿票不要在本周初用光。
  • 7/10 起源帝牙卢卡、起源帕路奇亚、原始盖欧卡、原始固拉多的价值最高，优先级高于普通三神鸟。
  • GO Fest 全球两天超级超梦 X / Y 分日登场：X 弱飞行 / 幽灵 / 妖精，Y 弱虫 / 幽灵 / 恶。

练哪类攻手

  • 短期最实用：岩石、电、冰、火、地面、格斗。它们覆盖三神鸟与超级路卡利欧。
  • 中期补强：龙、妖精、恶、幽灵。暗影帕路奇亚、奈克洛兹玛形态、超级超梦与 GO Fest 多传说都会用到。
  • Max 侧本周是极巨化豆豆鸽，低压力；真正值得投入的 Max 攻手仍看 Max 攻手榜 A/B Tier。

逐 Boss 克制与同属性 Mega 糖果搭配见「当前团战 Counter」。
