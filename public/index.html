本月看点

6 月下旬的节奏已经固定：凉脊龙社区日、飞行计程车连续活动、5★ / 超级团战周三轮换，以及 Max 周一接力到咕咕。

  • 6/20 凉脊龙 社区日：闪光概率提升、捕捉星尘 ×3、捕捉糖果 ×2；冰 / 龙系资源优先级最高。
  • 6/23–6/29 飞行计程车与6/25–6/29 被接管相互重叠，留意飞行系出现与火箭队相关限时调查。
  • 6/24 起 5★切到纸御剑 / 铁火辉夜，超级团战切到超级大比鸟；6/24 晚上还有对应团战时刻。
  • 6/27 盔甲鸟 超级团战日是本月另一个闪光重点窗口；火 / 电系队伍提前配好。
  • 7 月预告：7/4 泪眼蜥社区日与十周年庆典同日开启，十周年庆典有捕捉 XP ×4、捕捉星尘 ×4 与活动主题闪光重点。

本月 Weekly Rotations

本期优先级（6/18–6/24）

先把资源投向 6/20 凉脊龙社区日：捕捉星尘 ×3、糖果与糖果 XL 加成适合开星星碎片集中刷。团战侧当前是 奈克洛兹玛与超级巨钳螳螂，6/24 后切到纸御剑 / 铁火辉夜与超级大比鸟；详细 Counter 请看「当前团战 Counter」。

活动窗口

  • 6/18 18:00–19:00 小山猪聚焦时刻：转移糖果 ×2，适合提前打标签清盒。
  • 6/20 14:00–17:00 凉脊龙社区日：闪光、星尘、糖果与 XL 都集中在这个窗口。
  • 6/22 Max 周一：极巨化咕咕；当前周 Max Boss 是石丸子，适合测试草 / 水 / 钢 / 地面 / 格斗 Max 攻手。
  • 6/23 起飞行计程车，6/25 被接管叠加；6/27 盔甲鸟超级团战日优先留远程 / 本地票。

练哪类攻手

  • 当前奈克洛兹玛弱虫 / 幽灵 / 恶；超级巨钳螳螂双重弱火。
  • 6/24 后纸御剑双重弱火，铁火辉夜弱火 / 电；火系与电系投资都能连续复用。
  • 6/27 盔甲鸟超级团战日继续吃火 / 电队伍，火系优先级高。
  • 7/4 十周年庆典有捕捉 XP ×4 与星尘 ×4，建议提前准备幸运蛋、星星碎片与盒子空间。

Mega Booster：同属性糖果搭配

机制：保持一个超级进化宝可梦处于激活状态时，捕捉与该 Mega 共享属性的宝可梦会获得 +1 糖果，并有额外糖果 XL 机会；这与“进化获得糖果”无关。

超级胡地
共享超能力：捕捉奈克洛兹玛、铁哑铃时吃同属性糖果加成。
超级化石翼龙
共享岩石：捕捉菊石兽、化石盔、岩狗狗、化石翼龙、石丸子时吃同属性糖果加成。
超级水箭龟
共享水：捕捉菊石兽、化石盔、墨海马时吃同属性糖果加成。
超级巨钳螳螂
共享虫 / 钢：捕捉巨钳螳螂、铁哑铃、帝牙卢卡时吃同属性糖果加成。
超级暴飞龙
共享龙 / 飞行：捕捉赤面龙、帝牙卢卡、化石翼龙、天蝎、Hisuian Decidueye 时可用。
超级妙蛙花
共享草 / 毒：捕捉 Hisuian Decidueye 时用草属性共享；毒属性也可覆盖同属性目标。
超级路卡利欧
共享格斗 / 钢：捕捉飞腿郎、Hisuian Decidueye、铁哑铃、帝牙卢卡时可覆盖对应属性。
超级耿鬼
共享幽灵 / 毒：捕捉泥偶小人、Shadow Alolan Marowak 时吃幽灵共享糖果加成。
超级大比鸟
共享一般 / 飞行：捕捉多边兽、化石翼龙、天蝎与咕咕时可用。
